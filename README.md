# kota-agent-skills

Open-source Agent Skills used at Kota.

This repo follows the Agent Skills format (https://agentskills.io/) and is intended to be installable via skills.sh.

## Repo Layout

```text
skills/
  <skill-name>/
    SKILL.md
    scripts/        (optional)
    references/     (optional)
    assets/         (optional)
```

## Adding A Skill

See `AGENTS.md` for conventions and a checklist.
